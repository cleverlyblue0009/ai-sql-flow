## 🔧 **TROUBLESHOOTING**

### **Backend won't start?**
```bash
pip install -r requirements.txt
python -m uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

### **Frontend won't start?**
```bash
npm install
npm run dev
```

### **Neon effects not showing?**
- Clear browser cache
- Check console for errors (F12)
- Verify neon.css exists in `/src/styles/`
